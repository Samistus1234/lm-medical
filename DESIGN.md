# L&M Medical Solutions Design System

Adapted from Stripe's design system for a premium medical supply company.

## Visual Identity
- Clean, trustworthy, precision-focused
- Deep navy + medical blue + silver palette
- Conservative border-radius (4px-8px)
- Blue-tinted multi-layer shadows
- Weight 300 headlines, weight 400 UI elements

## Color Palette

### Primary
- **Medical Blue** `#1a6bb5` — Primary CTA, links, interactive
- **Medical Blue Hover** `#155a96` — Hover states
- **Deep Navy** `#0a1628` — Headings, strong text
- **Dark Navy** `#0d1b2a` — Dark sections, sidebar, footer
- **Secondary Blue** `#2a8fd4` — Accent, secondary actions
- **White** `#ffffff` — Backgrounds, card surfaces

### Text
- **Heading** `#0a1628` — Primary headings
- **Label** `#273951` — Form labels, secondary headings
- **Body** `#64748d` — Descriptions, captions
- **Muted** `#94a3b8` — Placeholder text

### Status
- **Success** `#15be53` — Success indicators
- **Success Text** `#108c3d` — Success badge text
- **Warning** `#f59e0b` — Warning states
- **Warning Text** `#9b6829` — Warning badge text
- **Error** `#ea2261` — Error states, destructive actions
- **Info** `#2a8fd4` — Info badges

### Surface & Borders
- **Border** `#e5edf5` — Default borders
- **Border Active** `#1a6bb5` — Active/focused borders
- **Surface** `#f8fafc` — Subtle backgrounds
- **Surface Hover** `#f1f5f9` — Hover backgrounds

### Shadows
- Ambient: `rgba(23,23,23,0.06) 0px 3px 6px`
- Standard: `rgba(23,23,23,0.08) 0px 15px 35px`
- Elevated: `rgba(50,50,93,0.25) 0px 30px 45px -30px, rgba(0,0,0,0.1) 0px 18px 36px -18px`
- Deep: `rgba(3,3,39,0.25) 0px 14px 21px -14px, rgba(0,0,0,0.1) 0px 8px 17px -8px`

## Typography
- **Primary font:** Inter (weight 300 headlines, 400 body/UI)
- **Monospace:** Source Code Pro
- **Display:** 48px, weight 300, letter-spacing -0.96px
- **Section Heading:** 32px, weight 300, letter-spacing -0.64px
- **Sub-heading:** 22px, weight 300, letter-spacing -0.22px
- **Body Large:** 18px, weight 300
- **Body:** 16px, weight 400
- **Caption:** 13px, weight 400
- **Micro:** 10px, weight 300

## Components
- **Border radius:** 4px (buttons, inputs, badges), 6px (cards, nav), 8px (featured)
- **Spacing:** 8px base unit
- **Buttons:** 8px 16px padding, 16px font, weight 400
- **Cards:** white bg, 1px solid #e5edf5, 6px radius, standard shadow

## Do's and Don'ts
- DO use `#0a1628` for headings, never pure black
- DO use blue-tinted shadows, never neutral gray
- DO keep border-radius 4-8px, never pill-shaped
- DO use weight 300 for headlines
- DON'T use bright/warm accents for primary actions
- DON'T use heavy font weights (600+) for headlines
